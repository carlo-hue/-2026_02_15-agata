📚 Informazioni

🔍 Box Least Squares (BLS)

Algoritmo ottimizzato per rilevare segnali periodici non-sinusoidali (transiti esopianeti). Cerca variazioni a forma di "box" nel flusso stellare.

Riferimento: Kovács et al. 2002, A&A 391, 369

📊 Metriche Rilevazione

  • SDE (Signal Detection Efficiency): (power - ⟨power⟩) / σ(power)
  • SDE > 7: Candidato robusto
  • SNR (Signal-to-Noise): Rapporto segnale/rumore
  • SNR > 5: Detection significativa

🌍 Parametri Fisici

  • Rp/R★: Rapporto raggi (da profondità = (Rp/R★)²)
  • a: Semiasse orbitale (Keplero III)
  • Teq: Temperatura equilibrio pianeta
  • Classe: Rocky / Super-Earth / Neptune / Jupiter
